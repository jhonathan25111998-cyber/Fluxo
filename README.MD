# FLUXO - Organize. Execute. Flua.

![Fluxo](logo.png)

## 📋 Sobre
FLUXO é um sistema de produtividade com:
- ✅ Metas e tarefas personalizáveis
- 📅 Calendário integrado
- 🍅 Timer Pomodoro para foco
- 🔐 Múltiplos usuários
- 🌙 Dark mode
- 💾 Backup e restauração de dados

## 🚀 Funcionalidades
- **Cards personalizáveis** - Crie, edite e organize suas categorias
- **Drag & Drop** - Organize tarefas arrastando entre cards
- **Recorrência** - Tarefas diárias, dias úteis, semanais e mensais
- **Calendário** - Visualize suas tarefas no calendário
- **Modo Foco** - Timer Pomodoro para concentração

## 🎯 Como usar
1. Crie uma conta ou use o usuário demo (`demo` / `demo123`)
2. Adicione seus cards e categorias
3. Crie suas metas e tarefas
4. Use o modo foco para produtividade

## 🛠️ Tecnologias
- HTML5
- CSS3 (Glassmorphism)
- JavaScript (Vanilla)
- LocalStorage para persistência

## 📱 Responsivo
Funciona em desktop, tablet e mobile!
